
# 環境構築方法
```
npm install
```

# 実行方法
```
// tsファイルのコンパイル
tsc lesson_file/1-1/test_1.ts

// jsファイルの実行
node lesson_file/1-1/test_1.js

// tsファイルをコンパイルせずに実行
npx ts-node lesson_file/1-1/test_1.ts
```

# テスト実行
```
npm t
```
もしくは `master` ブランチにプッシュすることで自動的にtestが実行される
